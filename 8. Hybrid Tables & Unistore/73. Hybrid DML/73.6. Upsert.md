# Upsert in Snowflake

**Description:**
An upsert operation in Snowflake combines the functionality of insert and update operations into one. It checks for the existence of a record based on a unique identifier and either inserts a new record or updates the existing one accordingly.

### Typical Scenario
Updating customer information in a CRM system where the customer record may or may not exist.

### Core Snowflake SQL Objects Used

### Code Source Execution
```sql
MERGE INTO customers AS target
USING (SELECT 'John Doe' AS name, 'johndoe@example.com' AS email) AS source
ON target.email = source.email
WHEN MATCHED THEN
  UPDATE SET name = source.name
WHEN NOT MATCHED THEN
  INSERT (name, email)
  VALUES (source.name, source.email);
```
