### Typical Scenario
Analyzing millions of raw transaction records to identify daily performance metrics by category.

### Core Snowflake SQL Objects Used

### Code Source Execution
```sql
```sql
  WITH raw_transactions AS (
      SELECT 
          transaction_id,
          DATE(transaction_timestamp) as transaction_date,
          product_category,
          sale_amount,
          customer_id
      FROM sales_db.transactions
      WHERE transaction_timestamp >= '2026-01-01'
  )
  SELECT 
      transaction_date,
      product_category,
      COUNT(DISTINCT customer_id) as unique_customers,
      COUNT(*) as total_transactions,
      SUM(sale_amount) as daily_revenue,
      AVG(sale_amount) as avg_transaction_size
  FROM raw_transactions
  GROUP BY 
      transaction_date, 
      product_category
  ORDER BY 
      transaction_date DESC, 
      daily_revenue DESC;
