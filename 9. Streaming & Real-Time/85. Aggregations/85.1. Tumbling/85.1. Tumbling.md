**Description:**

### Code Source Execution
```sql

  SELECT 
      TIME_SLICE(recorded_at, 10, 'SECOND', 'START') AS window_start,
      TIME_SLICE(recorded_at, 10, 'SECOND', 'END') AS window_end,
      ride_id,
      MAX(g_force) AS peak_g_force,
      AVG(rotation_speed) AS avg_rotation,
      COUNT(*) AS sensor_readings
  FROM coaster_sensors
  GROUP BY 
      TIME_SLICE(recorded_at, 10, 'SECOND', 'START'),
      TIME_SLICE(recorded_at, 10, 'SECOND', 'END'),
      ride_id
  
```
